# ⚡ ClipMaker — AI Viral Moments

Projeto desenvolvido durante o evento **NLW** da plataforma **Rocketseat**, com o objetivo de criar uma ferramenta que utiliza **Inteligência Artificial** para identificar e extrair os momentos mais virais de vídeos longos.  

O sistema permite o **upload de vídeos**, gera a **transcrição automática do áudio** e utiliza o **modelo Gemini da Google** para analisar o conteúdo e selecionar o trecho mais engajador para redes sociais.

O **ClipMaker** foi desenvolvido com foco em **performance**, **experiência do usuário (UX)** e **integrações modernas com APIs de IA e serviços de Cloud**.

## Funcionalidades
- Upload de vídeos com interface otimizada.
- Armazenamento e processamento de mídia na nuvem.
- Geração automática de transcrição do áudio do vídeo.
- Análise da transcrição com Inteligência Artificial.
- Identificação automática do trecho mais viral do vídeo.
- Corte inteligente do vídeo baseado no tempo sugerido pela IA.
- Interface moderna com animações suaves e modo escuro.
